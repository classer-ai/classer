# Classer

Low-cost, fast AI classification API. 10x cheaper than GPT, <200ms latency.

## SDKs

- [Python SDK](./python) — `pip install classer`
- [TypeScript SDK](./typescript) — `npm install classer-ai`

## Documentation

See [docs.classer.ai](https://docs.classer.ai) or the [docs](./docs) folder.
