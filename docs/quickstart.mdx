## 1. Get your API key

Create one at [classer.ai/api-keys](https://classer.ai/api-keys).

## 2. Install the SDK

<CodeGroup>

```bash Python
pip install classer
```

```bash TypeScript
npm install classer-ai
```

</CodeGroup>

## 3. Classify text

Pick the single best label for a piece of text.

<CodeGroup>

```python Python
import classer
import os

os.environ["CLASSER_API_KEY"] = "your-api-key"

result = classer.classify(
    text="I need help resetting my password",
    labels=["billing", "technical", "sales"]
)

print(f"Label: {result.label}")           # "technical"
print(f"Confidence: {result.confidence}")  # 0.94
```

```typescript TypeScript
import { classify } from "classer-ai";

process.env.CLASSER_API_KEY = "your-api-key";

const result = await classify({
  text: "I need help resetting my password",
  labels: ["billing", "technical", "sales"]
});

console.log(`Label: ${result.label}`);           // "technical"
console.log(`Confidence: ${result.confidence}`);  // 0.94
```

```bash cURL
curl -X POST https://api.classer.ai/v1/classify \
  -H "Authorization: Bearer $CLASSER_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "I need help resetting my password",
    "labels": ["billing", "technical", "sales"]
  }'
```

</CodeGroup>

## 4. Tag with multiple labels

Return all labels that match above a confidence threshold.

<CodeGroup>

```python Python
result = classer.tag(
    text="Breaking: Tech stocks surge amid AI boom",
    labels=["politics", "technology", "finance", "sports"],
    threshold=0.3
)

for t in result.labels:
    print(f"{t.label}: {t.confidence}")
# technology: 0.92
# finance: 0.78
```

```typescript TypeScript
import { tag } from "classer-ai";

const result = await tag({
  text: "Breaking: Tech stocks surge amid AI boom",
  labels: ["politics", "technology", "finance", "sports"],
  threshold: 0.3
});

for (const t of result.labels) {
  console.log(`${t.label}: ${t.confidence}`);
}
// technology: 0.92
// finance: 0.78
```

```bash cURL
curl -X POST https://api.classer.ai/v1/tag \
  -H "Authorization: Bearer $CLASSER_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "Breaking: Tech stocks surge amid AI boom",
    "labels": ["politics", "technology", "finance", "sports"],
    "threshold": 0.3
  }'
```

</CodeGroup>

## 5. Use descriptions for better accuracy

Add label descriptions when labels are ambiguous.

<CodeGroup>

```python Python
result = classer.classify(
    text="We need a solution for 500 users, what's your enterprise pricing?",
    labels=["hot", "warm", "cold"],
    descriptions={
        "hot": "Ready to buy, asking for pricing or demo",
        "warm": "Interested but exploring options",
        "cold": "Just browsing, no clear intent"
    }
)

print(result.label)  # "hot"
```

```typescript TypeScript
const result = await classify({
  text: "We need a solution for 500 users, what's your enterprise pricing?",
  labels: ["hot", "warm", "cold"],
  descriptions: {
    hot: "Ready to buy, asking for pricing or demo",
    warm: "Interested but exploring options",
    cold: "Just browsing, no clear intent"
  }
});

console.log(result.label);  // "hot"
```

</CodeGroup>
